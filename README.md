# ObsercormaLang

ObsercormaLang est un langage expérimental et un moteur de transformation de texte écrit en Python.
Il applique des règles spécifiques de modification de chaînes de caractères afin de produire une forme alternative et non conventionnelle de langage, destinée à l’expérimentation, au jeu linguistique et à la recherche créative.

> [!IMPORTANT]
> Version de Python 3.13
